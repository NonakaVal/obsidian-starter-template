# üóÇ ACE Organization System

The **ACE** (Atlas, Calendar, Efforts) framework is a universal system for organizing **knowledge, time, and action**, complemented by:

> How it works ‚Üí Atlas ¬∑ Calendar ¬∑ Efforts  

![img](https://imgur.com/I3qZKux.png)
### Components
- **A ‚Äì Atlas** ‚Üí Stores references and general knowledge. It‚Äôs the *map* of your ideas, concepts, and topics.  
- **C ‚Äì Calendar** ‚Üí Holds everything linked to dates: events, deadlines, chronological records.  
- **E ‚Äì Efforts** ‚Üí Contains your projects, tasks, and initiatives that require action.

# üåà ARC Framework

The **ARC Framework** defines the flow of your ideas through time and space, enhancing your creative process.  

**Cycle:** New ideas ‚Üí Natural organization ‚Üí Concrete application = **Self-sustaining learning system**  

> ARC = Add ¬∑ Relate ¬∑ Communicate

# üè∑ Metadata

### Main Metadata Fields
For better control, **Metadata** is used, such as:  

- `created` ‚Üí creation date  
- `up` ‚Üí hierarchy  
- `collection` ‚Üí belonging collection  
- `related` ‚Üí associated notes
